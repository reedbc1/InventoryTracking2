# Inventory Tracking

## Description
Expands on InventoryTacking repo. Data cleaning, exploratory analysis, and predictive modeling for diaper bank distribution center inventory.

## Creating a Local Virtual Environment

### Step 1. Create environment folder with venv  
`python -m venv .venv`

### Step 2. Activate the virtual environment
For Windows / PowerShell:   
`PS C:\> .venv\Scripts\Activate.ps1`  

For Linux / Bash:  
`$ source .venv/bin/activate`

### Step 3. Install Dependencies
Upgrade pip  
`pip install --upgrade pip`

Install from requirements.txt  
`pip install -r requirements.txt`

## Running Jupyter Notebooks
To run the files in the `notebooks` folder, you will need to have Jupyter
installed or install the Jupyter extension from VSCode.
